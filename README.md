A comprehensive, full-stack flight reservation system connecting the major regions of Solaris-3. 
Built as a combined Information Management and Data Structures & Algorithms project, 
*SFN* handles dynamic flight scheduling, tiered seat bookings (Economy to First Class), and in-flight service management. 🌍✈️
## 🚀 Features

- 📈 **Dynamic Algorithmic Pricing** -- Utilizes custom algorithms (DSA) to calculate and adjust real-time seat prices based on demand, capacity, and cabin tier.
  
- 🗺️ **Smart Connecting Flights** -- Efficiently routes passengers across multiple Solaris-3 regions, handling complex multi-leg journeys.
  
- 💺 **Interactive Seat Selection** -- Visual booking interface allowing users to reserve specific seats within Economy, Business, or First-Class cabins.
  
- 🧳 **Comprehensive Add-on Management** -- Robust database integration for handling custom user requests, including in-flight Wi-Fi, catered meals, and extra baggage allowances.
  
- ⚙️ **Combined System Architecture** -- Seamlessly merges Information Management (relational databases) and Data Structures & Algorithms into a single, cohesive web application.

## 🛠️ Setup & Running

### **Prerequisites**
-   Java 17+
-   MySQL Server
-   VS Code (with Extension Pack for Java)
-   Git

### **Installation**

**1. Clone the repository**
```bash
git clone https://github.com/frostishyper/Solaris-Flight-Network.git
cd Solaris-Flight-Network
```
